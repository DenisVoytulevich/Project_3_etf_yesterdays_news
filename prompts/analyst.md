## Формат ответа
Верни JSON с ключами ниже. Каждое значение — готовый **markdown** (без code block).

Во **всех таблицах** для порядкового номера используй колонку **`#`** (не «№»).

### executive_summary
2–3 предложения: общий тон дня, главный драйвер, на что смотреть.

### top_market_news
**§1** — таблица:

| # | Событие | Сила | Сектор | Драйвер сектора |
|---|---------|------|--------|-----------------|

- **#** — номер события (1, 2, 3…); **одинаковый** для всех строк одного события.
- **Событие** — факт на русском, до 12 слов, только вчера; повторяй в каждой строке события.
- **Сила** — целое **−5…+5** по единой шкале (одинаковое для всех строк одного события).
- **Сектор** — один на строку, английский.
- **Драйвер сектора** — `{драйвер}: {изменение}`, до 10 слов. Для убытков/рисков/страховых выплат **рост** драйвера = негатив для сектора.
- 8–20 строк суммарно.

### sector_ratings
**§2** — таблица:

| # | Отрасль | Влияние | Обоснование |
|---|---------|---------|-------------|

- **#** — порядковый номер строки.
- **Отрасль** — **точно** как в блоке «Отрасли интереса» (без перевода и синонимов).
- **Влияние** — целое **−5…+5** по единой шкале; все отрасли из «Отрасли интереса».
- 10–15 строк, сортировка по влиянию (от +5 к −5).

### portfolio_companies_news
**§3** — таблица:

| Компания | Зона | Отрасль | Новость за вчера | Влияние |
|----------|------|---------|------------------|---------|

- Только компании из списка.
- **Влияние** — **только число** **−5…+5** (не слова «высокая/средняя/низкая/низкое»).
- 5–15 строк или строка-заглушка.

### key_risks_today
**§4** — таблица:

| Время | Событие | Тип | Влияние | На что влияет |
|-------|---------|-----|---------|---------------|

- Только события **сегодня** ({{ trading_date }}).
- Тип: Макро / ЦБ / Отчёт / Геополитика / Регуляторика.
- **Влияние** — целое **−5…+5** по единой шкале.
- 5–12 строк.

## Согласованность
1. §1 — факты + «Сила» + текстовый драйвер сектора.
2. §2 агрегирует §1; все отрасли интереса в таблице.
3. §3 — только компании из списка, новости за вчера.
4. §4 — только сегодня, без дублей §1 и §3.
5. Во всех секциях — **одна числовая шкала** (−5…+5).

Без рекомендаций «покупай/продавай». Не выдумывай события вне источников.
